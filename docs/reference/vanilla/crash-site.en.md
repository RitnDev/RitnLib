# `lualib/vanilla/crash-site`


A **copy** of Factorio's engine `crash-site` file (the starting crash-scene generation).

| | |
|---|---|
| **Source** | `lualib/vanilla/crash-site.lua` |
| **Origin** | fork of `__base__` (crash-site) |
| **Access** | `require(ritnlib.defines.vanilla.crash_site)` |

> **Nature** — A copy of the engine file with functions made **local** (optimization). Same API as the vanilla crash-site — see the [official source](https://github.com/wube/factorio-data). Excluded from LuaLS annotations, not re-documented here.
