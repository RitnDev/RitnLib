# `lualib/vanilla/util`


**Recopie** du fichier `util` du moteur Factorio.

| | |
|---|---|
| **Source** | `lualib/vanilla/util.lua` |
| **Origine** | fork de `__core__/lualib/util` |
| **Accès** | `require(ritnlib.defines.vanilla.util)` |

> **Nature** — Ces fichiers `vanilla/` sont des **recopies des fichiers moteur** dont les fonctions sont rendues **locales** (au lieu de globales comme dans le jeu), ce qui les rend **plus optimisées** à l'usage. L'API publique reste celle du `util` vanilla de Factorio — se référer à la [doc/source officielle](https://github.com/wube/factorio-data) pour le détail des fonctions.
>
> Volontairement **exclu des annotations LuaLS** de RitnLib (comme tout `vanilla/`), donc non re-documenté fonction par fonction ici.
